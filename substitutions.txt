# Lunar Eternal Blue Un-Worked Design patch substitution file
# Format:
# original | replacement;
# The spaces before and after the "|" are mandatory.
#
# When substituting, the parser will try replacing any space
# characters in the input with periods, newlines, etc., to try to
# catch as many uses as possible.
#
# IMPORTANT: Strings are substituted in the order they are
# listed in the file. If a shorter strings precedes a longer one
# (e.g. "Althena" comes before "Goddess Althena"), the longer subtitution
# will never occur because the shorter one will occur first.
#=============================
